# The Idea
Creating a personal website for a friend who is a photographer, showcasing: <br>
- Portfolio and galleries <br>
- About section <br>
- Contact information <br>
- A clean, modern visual identity

---

# How It Started
I began by choosing a **W3Schools W3.CSS template** as a base layout. <br>

Steps: <br>
- Tested the template using the W3Schools online editor <br>
- Modified structure, content and layout directly in the browser <br>
- Later decided to move the entire project to **Visual Studio Code** <br>
- Started organizing files, assets and libraries properly <br>

---

# Moving to VS Code
Why switch to VS Code? <br>
- Better file structure control <br>
- Easier CSS and JavaScript management <br>
- Ability to use extensions (Live Server) <br>
- Local image and asset preview, while W3Schools free version had limit on numbers of pictures I could upload. 


---
