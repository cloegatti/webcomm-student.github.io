The Idea

Creating a personal website for a friend who is a photographer, showcasing:
- Portfolio and galleries
- About section
- Contact information
- A clean, modern visual identity

How It Started

I began by choosing a W3Schools W3.CSS template as a base layout.

Steps:
- Tested the template using the W3Schools online editor
- Modified structure, content and layout directly in the browser
- Later decided to move the entire project to Visual Studio Code
- Started organizing files, assets and libraries properly

Moving to VS Code

Why switch to VS Code?
- Better file structure control
- Easier CSS and JavaScript management
- Ability to use extensions (Live Server)
- Local image and asset preview, while W3Schools free version had limit on numbers of pictures I could upload.

Template Customization

Once imported into VS Code, I started customizing the template:
- Modified classes and layout from the original W3.CSS structure
- Added personalized color palette and typography
- Reorganized the gallery to better match the photographerâ€™s style
- Implemented responsive adjustments

CSS Personalization

I wrote custom CSS rules to override the default W3.CSS, which was basically already set through a live server precompiled sheet to my own style.

JavaScript Additions

I added lightweight JavaScript functions to improve usability:

  • Modal image viewer
  • Smooth scroll for navigation
  • Button back to the top
  • About me section effect

Future of the Website

The website will be implemented in future adding new pictures as soon as the photographer takes new one. It would be nice to add more pages looking deeply in any style he works on.
